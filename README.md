# Compound Interest Calculator

A simple Python program that calculates the future value of an investment using the compound interest formula.

## Features
- User input for principal, interest rate, and number of periods
- Calculates future value
- Formats output to two decimal places

## Example

Principal/Face Value: 1000
Periodic Interest Rate (%): 5
Number of periods: 3

Future Value: 1157.63

## Concepts Practiced
- Functions
- User input
- f-strings
- Arithmetic operations
