# python-finance-foundations
Python fundamentals and introductory finance projects completed as part of a quant finance learning roadmap.

## Projects

### Week 1 — Compound Interest Calculator
Calculates the future value of an investment using compound interest.

### Week 2 — Portfolio Tracker
Loads holdings from a CSV file and calculates portfolio weights.

### Week 3 — Portfolio Variance Calculator
Uses NumPy matrix operations to compute portfolio variance.

## Skills Covered
- Python fundamentals
- Functions and classes
- File I/O
- NumPy
